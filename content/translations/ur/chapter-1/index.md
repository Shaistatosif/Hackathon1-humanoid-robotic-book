# باب 1: ہیومنائیڈ روبوٹکس کا تعارف

## سیکھنے کے مقاصد

اس باب کے اختتام تک، آپ یہ کر سکیں گے:

- ہیومنائیڈ روبوٹ کی تعریف کریں
- ہیومنائیڈ روبوٹکس کی تاریخی ترقی کی وضاحت کریں
- ہیومنائیڈ روبوٹس کو دوسری اقسام سے ممتاز کرنے والی اہم خصوصیات کی شناخت کریں
- ہیومنائیڈ روبوٹس کی اہم ایپلیکیشنز اور استعمال کے معاملات کو سمجھیں

## 1.1 ہیومنائیڈ روبوٹ کیا ہے؟

**ہیومنائیڈ روبوٹ** ایک ایسا روبوٹ ہے جو شکل اور فنکشن میں انسانی جسم کی نقل کرنے کے لیے ڈیزائن کیا گیا ہے۔ صنعتی روبوٹس کے برعکس جو مخصوص کاموں کے لیے موزوں ہیں، ہیومنائیڈ روبوٹس انسانی ماحول میں کام کرنے اور لوگوں کے ساتھ قدرتی طور پر بات چیت کرنے کے لیے بنائے گئے ہیں۔

> **تعریف:** ہیومنائیڈ روبوٹ ایک خودمختار یا نیم خودمختار مشین ہے جس کی جسمانی ساخت انسانی شکل پر مبنی ہے، عام طور پر سر، دھڑ، دو بازو، اور دو ٹانگیں ہوتی ہیں۔

### اہم خصوصیات

ہیومنائیڈ روبوٹس میں کئی مشترک خصوصیات ہیں:

| خصوصیت | تفصیل |
|--------|--------|
| **دو پیروں پر چلنا** | انسانوں کی طرح دو ٹانگوں پر چلنا |
| **انسان نما ڈیزائن** | انسان جیسی جسمانی تناسب اور ظاہری شکل |
| **مہارت سے ہیرا پھیری** | چیزوں کو پکڑنے اور ہیرا پھیری کرنے کی صلاحیت |
| **حسی ادراک** | بینائی، سماعت، لمس، اور توازن کا احساس |
| **سماجی تعامل** | انسانوں کے ساتھ بات چیت اور تعامل کی صلاحیت |

### آزادی کی ڈگریاں

انسانی جسم میں تقریباً 244 degrees of freedom (DOF) ہیں۔ جدید ہیومنائیڈ روبوٹس عام طور پر 20-50 DOF نافذ کرتے ہیں:

```python
# مثال: ایک عام ہیومنائیڈ روبوٹ میں آزادی کی ڈگریاں
humanoid_dof = {
    "head": {
        "neck_pan": 1,      # بائیں-دائیں گردش
        "neck_tilt": 1,     # اوپر-نیچے گردش
        "neck_roll": 1,     # سائیڈ جھکاؤ
    },
    "arm": {  # ہر بازو کے لیے
        "shoulder_pitch": 1,
        "shoulder_roll": 1,
        "shoulder_yaw": 1,
        "elbow": 1,
        "wrist_roll": 1,
        "wrist_pitch": 1,
        "wrist_yaw": 1,
    },
    "torso": {
        "waist_yaw": 1,
        "waist_pitch": 1,
    },
    "leg": {  # ہر ٹانگ کے لیے
        "hip_pitch": 1,
        "hip_roll": 1,
        "hip_yaw": 1,
        "knee": 1,
        "ankle_pitch": 1,
        "ankle_roll": 1,
    }
}

def calculate_total_dof(config):
    """کنفیگریشن سے کل آزادی کی ڈگریوں کا حساب لگائیں۔"""
    total = 0
    for part, joints in config.items():
        if part in ["arm", "leg", "hand"]:
            total += sum(joints.values()) * 2
        else:
            total += sum(joints.values())
    return total

total_dof = calculate_total_dof(humanoid_dof)
print(f"کل DOF: {total_dof}")  # آؤٹ پٹ: کل DOF: 44
```

## 1.2 تاریخی ترقی

ہیومنائیڈ روبوٹس کا تصور صدیوں میں نمایاں طور پر تبدیل ہوا ہے:

### قدیم آغاز (1900 سے پہلے)

مصنوعی انسان بنانے کا خواب قدیم تہذیبوں سے ہے:

- **قدیم یونان:** Talos کی داستانیں، ایک دیو کانسی کا automaton
- **قرون وسطیٰ:** مکینیکل نائٹس اور گھڑیوں میں خودکار شخصیات
- **18ویں صدی:** Jacques de Vaucanson کی "Digesting Duck" اور دیگر automata

### جدید ہیومنائیڈ دور (1970-حال)

| سال | روبوٹ | اہمیت |
|-----|-------|-------|
| 1986 | Honda E0 | Honda کے دو پیروں والے روبوٹس کا پہلا |
| 2000 | ASIMO | ترقی یافتہ دو پیروں کی حرکت |
| 2013 | Atlas | Boston Dynamics، متحرک توازن |
| 2023 | Figure 01 | کام کی جگہ کے لیے تجارتی ہیومنائیڈ |

## 1.3 ہیومنائیڈ روبوٹس کی درجہ بندی

### ایپلیکیشن کے لحاظ سے

1. **ریسرچ پلیٹ فارمز:** تعلیمی تحقیق کے لیے (مثلاً iCub، NAO)
2. **صنعتی/تجارتی:** کام کی جگہ کے کاموں کے لیے (مثلاً Atlas، Figure)
3. **سماجی/سروس:** انسانی تعامل پر توجہ (مثلاً Pepper، Sophia)
4. **تفریح:** تھیم پارکس اور مظاہرے

## 1.4 ہیومنائیڈ روبوٹس کی ایپلیکیشنز

### صحت کی دیکھ بھال اور مدد

ہیومنائیڈ روبوٹس صحت کی دیکھ بھال کی ترتیبات میں تیزی سے استعمال ہو رہے ہیں:

- **مریض کی دیکھ بھال:** بزرگ اور معذور افراد کی مدد
- **بحالی:** فزیکل تھراپی سپورٹ اور مانیٹرنگ
- **رفاقت:** تنہائی کو کم کرنا اور سماجی تعامل فراہم کرنا

### صنعتی ایپلیکیشنز

- **مینوفیکچرنگ:** انسانی ڈیزائن کردہ جگہوں میں اسمبلی کے کام
- **لاجسٹکس:** گودام کے آپریشنز اور پیکیج ہینڈلنگ
- **تعمیرات:** خطرناک یا مشکل جگہوں میں کام

## خلاصہ

ہیومنائیڈ روبوٹس روبوٹکس کے سب سے زیادہ مہتواکانکشی اہداف میں سے ایک کی نمائندگی کرتے ہیں - ایسی مشینیں بنانا جو انسانی ماحول میں بغیر کسی رکاوٹ کے کام کر سکیں۔

اس باب کے اہم نکات:

- ہیومنائیڈ روبوٹس کی تعریف ان کی انسان جیسی شکل اور انسانی ماحول میں کام کرنے کی صلاحیت سے ہوتی ہے
- یہ شعبہ قدیم automata سے جدید ترقی یافتہ پلیٹ فارمز تک ارتقا پذیر ہوا ہے
- ایپلیکیشنز صحت کی دیکھ بھال، صنعت، تحقیق، اور آفات کے ردعمل میں پھیلی ہوئی ہیں

2. ہیومنائیڈ روبوٹکس کی تاریخ میں تین اہم سنگ میل بیان کریں۔
