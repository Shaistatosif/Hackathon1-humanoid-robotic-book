# Demo Video Script - Humanoid Robotics Textbook
**Duration:** 90 seconds
**Recording Tool:** Loom, OBS, or Windows Game Bar (Win + G)

---

## Pre-Recording Checklist
- [ ] Browser: Chrome/Edge (clean, no extra tabs)
- [ ] Open: https://hackathon-one-humanoid-robotic-book.vercel.app
- [ ] Screen resolution: 1920x1080 (recommended)
- [ ] Test login ready (create account before recording)
- [ ] Microphone working

---

## SCRIPT

### Scene 1: Intro (0:00 - 0:08)
**Screen:** Homepage with logo visible

**Voiceover:**
> "Meet our AI-Native Humanoid Robotics Textbook - an interactive learning platform that makes robotics education accessible to everyone."

**Action:** Slow scroll to show the homepage

---

### Scene 2: Browse Content (0:08 - 0:20)
**Screen:** Click on "Chapter 1: Introduction to Humanoid Robotics"

**Voiceover:**
> "Students can browse beautifully formatted chapters with professional typography, code examples, and clear explanations."

**Action:**
1. Click Chapter 1
2. Scroll through content slowly
3. Show a code block or diagram

---

### Scene 3: AI Chatbot (0:20 - 0:40)
**Screen:** Click the chat widget (bottom right corner)

**Voiceover:**
> "Our RAG-powered AI chatbot answers questions using the textbook content. Let me ask: What is a humanoid robot?"

**Action:**
1. Click chat button
2. Type: "What is a humanoid robot?"
3. Wait for response
4. Show the answer with citations

**Voiceover (when answer appears):**
> "Notice how it provides accurate answers with chapter citations - no hallucinations, just facts from the textbook."

---

### Scene 4: Urdu Translation (0:40 - 0:55)
**Screen:** Click language toggle (navbar)

**Voiceover:**
> "For accessibility, the entire platform supports Urdu with proper right-to-left layout."

**Action:**
1. Click language dropdown
2. Select "اردو" (Urdu)
3. Show RTL layout
4. Scroll to show Urdu content
5. Point out code blocks remain in English

**Voiceover:**
> "Technical code stays in English while explanations are fully translated."

---

### Scene 5: Quiz Feature (0:55 - 1:15)
**Screen:** Navigate to Quiz page

**Voiceover:**
> "Students can test their knowledge with AI-generated quizzes for each chapter."

**Action:**
1. Click "Quiz" in navbar
2. Select Chapter 1 Quiz
3. Answer 1-2 questions quickly
4. Submit and show score

**Voiceover (when score appears):**
> "Instant feedback with detailed scoring helps students track their progress."

---

### Scene 6: Dashboard (1:15 - 1:25)
**Screen:** Navigate to Dashboard

**Voiceover:**
> "The personalized dashboard shows reading progress, quiz history, and smart recommendations."

**Action:**
1. Click "Dashboard"
2. Show progress cards
3. Show quiz history
4. Show recommendations section

---

### Scene 7: Outro (1:25 - 1:30)
**Screen:** Return to homepage

**Voiceover:**
> "Humanoid Robotics Textbook - AI-powered learning for the next generation of roboticists."

**Action:** Show logo and homepage

---

## Recording Tips

1. **Pace:** Speak slowly and clearly
2. **Mouse:** Move cursor smoothly, don't rush clicks
3. **Pause:** Wait for pages to load before speaking
4. **Mistakes:** If you make a mistake, pause and re-record that section
5. **Background:** Close notifications, use Do Not Disturb mode

## Post-Recording

1. Trim start/end
2. Add background music (optional, keep low volume)
3. Export as MP4 (1080p)
4. Verify duration is under 90 seconds

If you prefer no voiceover:

## Quick Recording with Loom

1. Go to https://loom.com
2. Sign up (free)
3. Install browser extension
4. Click Loom icon → "Screen + Cam" or "Screen Only"
5. Select "Current Tab"
6. Click Record
7. Follow the script above
8. Click Stop when done
9. Download MP4
